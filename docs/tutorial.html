Guided workflow 引导流程

How to use Penguin from an empty tab to a working request. 从空白 tab 到成功发出一条请求，Penguin 怎么用。

This follows the same mental model as the in-app interactive tutorial, but in a static reference format. 这页按 app 里的 interactive tutorial 逻辑整理，但用静态文档方式呈现。

1. Packages sidebar 1. Packages 侧边栏

The sidebar is where installed packages, services, and methods appear. This is your API browser. 侧边栏会显示已安装的 packages、services、methods。这里就是你的 API 浏览器。

2. Install a package 2. 安装 package

Open the installer with 按 Cmd + S and install one of these types: 打开安装器，然后可安装这些类型：

  • @snsoft/*-grpc-web@version
  • @snsoft/*-grpc@version
  • @snsoft/js-sdk@version

After installation, Penguin parses proto files or typings and builds the service tree automatically. 安装后，Penguin 会自动解析 proto 或 typings，生成 service tree。

3. Tabs and protocols 3. Tabs 和协议

Each tab is an independent workspace. You can cycle protocol with Cmd + E and open more tabs with Cmd + N. 每个 tab 都是独立工作区。你可以用 Cmd + E 切协议，用 Cmd + N 开新 tab。

4. URL and environments 4. URL 和环境

The URL bar supports environment interpolation like {{URL}}. Each protocol keeps its own environments and active selection. URL bar 支持像 {{URL}} 这样的环境变量插值。每个协议都有独立环境和当前选择。

5. Request editor 5. 请求编辑器

Edit request headers and JSON body in the left panel. If Penguin knows the request schema, it can auto-fill a default body for you. 左边面板用来编辑 request headers 和 JSON body。如果 Penguin 知道 schema，就能自动填默认 body。

  • Format body 格式化 body
  • Reset to default 重置成默认值
  • Copy raw request body 复制原始请求 body

6. Search and select a method 6. 搜索并选择 method

Use Cmd + F to search services and methods across the current package set, then load the method into the active tab. 用 Cmd + F 搜索 services 和 methods，然后把方法载入当前 tab。

7. Send and inspect 7. 发送并查看

Send with Cmd + Enter. If the request is in flight, cancel with Esc. The right panel shows status, timing, headers, and response body. 用 Cmd + Enter 发送；发送中可用 Esc 取消。右边面板会显示状态、耗时、headers 和响应 body。

8. History, saved requests, and reuse 8. 历史、保存请求、复用

Save the current request with Cmd + Shift + S, open history with Cmd + H, and open saved requests with Cmd + O. 用 Cmd + Shift + S 保存当前请求，用 Cmd + H 打开历史，用 Cmd + O 打开 saved requests。

9. Built-in tools 9. 内建工具

  • Cmd + D for request documentation 查看请求文档
  • Cmd + P for proto definitions 查看 proto 定义
  • Cmd + I for network check and speed test 做网络检查和测速
  • Cmd + Shift + I to import from cURL 从 cURL 导入
