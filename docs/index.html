macOS blocked the app on first launch (Gatekeeper). What to do?

This is normal for apps downloaded outside App Store.

  • Option A (fast): right-click the app → Open → confirm.
  • Option B: System Settings → Privacy & Security → find “blocked app” → Open Anyway.

Tip: If you plan to distribute publicly, consider notarization to reduce blocks.

Which download do I pick?
  • Apple Silicon: M1 / M2 / M3 / M4 Macs → download aarch64.
  • Intel Macs → download x86_64.
Do you support Windows / Linux?

Not yet. Current builds target macOS only.
