Penguin is a desktop client for package-driven API testing across gRPC-Web, native gRPC, and SDK workflows. Penguin 是一个桌面 API client，重点是 package 驱动的 gRPC-Web、原生 gRPC、SDK 测试流程。

UI structure 界面结构

Sidebar 侧边栏

Installed packages, service tree, and method selection. 已安装 packages、service tree、method 选择。

Tab bar 标签栏

Independent workspaces with protocol state per tab. 每个 tab 都是独立工作区，协议状态各自保存。

URL bar URL 栏

Target URL with environment interpolation support. 目标 URL，支持环境变量插值。

Split panels 左右面板

Left side for request editing, right side for response inspection. 左边编辑请求，右边查看响应。

Protocols 协议

gRPC-Web

Uses ConnectRPC plus a Rust HTTP proxy. 使用 ConnectRPC，再加 Rust HTTP proxy。

gRPC

Uses a Node sidecar for native gRPC execution. 通过 Node sidecar 跑原生 gRPC。

SDK

Loads and executes SDK methods from a package bundle. 从 package bundle 载入并执行 SDK 方法。

Packages and discovery Packages 和解析

  • Packages are installed per protocol under Packages 会按协议安装在 ~/.pengvi/<protocol>/
  • Supported types: 支持类型： @snsoft/*-grpc-web, @snsoft/*-grpc, @snsoft/js-sdk
  • Proto files, connect typings, and SDK typings are parsed into a service/method model. Proto、connect typings、SDK typings 会被解析成 service/method model。

Environments and headers 环境和 headers

  • Each protocol keeps its own environment list and active selection. 每个协议都有自己的 environment list 和当前选择。
  • Variables use 变量格式是 {{VAR}}
  • Default headers are configurable per protocol. 默认 headers 可以按协议分别设置。

Built-in tools 内建工具

  • Request documentation dialog 请求文档对话框
  • Proto viewer for request/response definitions 查看请求/响应定义的 proto viewer
  • Network check and speed test 网络检查和测速
  • Import from cURL 从 cURL 导入
  • Copy current request as cURL 把当前请求复制成 cURL

Settings and persistence 设置和存储

  • Themes, display name, max history size, default headers. 主题、显示名称、历史容量、默认 headers。
  • Export/import environments, history, saved requests, and defaults. 导出/导入环境、历史、保存请求和默认值。
  • Local UI state is persisted in localStorage. 本地 UI 状态保存在 localStorage。

Config file 配置文件

A config file can predefine environments and packages. 配置文件可以预先定义 environments 和 packages。 

{
  "grpc-web": {
    "environments": [
      {
        "name": "UAT",
        "color": "amber",
        "variables": {
          "URL": "https://example.com",
          "TOKEN": ""
        }
      }
    ],
    "packages": ["@snsoft/player-grpc-web@1.2.3"]
  }
}

Open default config in repo 打开 repo 里的默认配置

Updates 更新

Penguin supports in-app update checks and installation through Tauri updater. Penguin 支持通过 Tauri updater 在 app 内检查更新和安装更新。
